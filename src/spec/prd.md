# WiFi QR 코드 카드 생성 서비스 PRD

## 개요
매장에서 출력하여 사용할 WiFi QR 코드 카드를 생성하는 웹 서비스입니다. 사용자가 브랜드 정보와 WiFi 정보를 입력하면 미니멀한 디자인의 QR 코드 카드를 생성하고 이미지로 다운로드할 수 있습니다.

## 주요 기능

### 1. QR 코드 카드 배경 커스터마이징
- 사용자가 카드 배경 색상을 선택할 수 있어야 함
- 다양한 색상 옵션 제공
- 실시간으로 배경 변경 미리보기

### 2. QR 코드 카드 이미지 다운로드
- 생성된 QR 코드 카드를 이미지 파일로 다운로드 가능
- 고품질 이미지 출력 지원
- 다양한 이미지 형식 지원 (PNG, JPG 등)

### 3. 미니멀한 디자인
- 깔끔하고 심플한 UI/UX
- 불필요한 요소 제거
- 직관적인 사용자 인터페이스

### 4. 입력 필드
- **브랜드 이름**: 카드 하단에 표시될 브랜드명
- **네트워크 이름**: WiFi 네트워크 SSID
- **비밀번호**: WiFi 접속 비밀번호
- **카드 배경 색**: QR 코드 카드의 배경 색상

### 5. 카드 레이아웃
- **상단**: "WIFI 접속" 텍스트
- **중앙**: QR 코드 (WiFi 정보 포함)
- **하단**: 브랜드 이름
- **하단 우측**: "by lian" 레이블 (작은 크기)

### 6. 카드 사이즈 및 비율
- **크기**: 256px 기준
- **비율**: 가로:세로 = 3:4
- **실제 크기**: 192px × 256px (3:4 비율)

## 기술 요구사항

### 프론트엔드
- Next.js 기반 웹 애플리케이션
- TypeScript 사용
- react-hook-form을 활용한 폼 관리
- 반응형 디자인
- 실시간 미리보기 기능

### QR 코드 생성
- WiFi 정보를 포함한 QR 코드 생성
- 표준 WiFi QR 코드 형식 준수
- 고품질 QR 코드 렌더링

### 이미지 다운로드
- Canvas API를 활용한 이미지 생성
- 브라우저 호환성 고려
- 다운로드 파일명 커스터마이징

## 사용자 시나리오

1. 사용자가 웹사이트에 접속
2. 브랜드 이름, 네트워크 이름, 비밀번호, 배경 색상 입력
3. 실시간으로 QR 코드 카드 미리보기 확인
4. 만족스러운 디자인이 완성되면 이미지 다운로드
5. 다운로드한 이미지를 인쇄하여 매장에 부착
